updating to new rails version:
* rvm update
* rvm reload
* rvm install 1.9.2

updating the app:
* bundle install
* gem update

updating code:
* bzr update

migrating db:
* rake db:migrate

running a server/console:
* bundle exec rails s
* ./script/rails c

running the tests:
* RAILS_ENV=test rake db:schema:load
* rake cucumber:wip
* bundle exec cucumber -f pretty -p wip features/transcription.feature:24 -b

running transcoding:
* rake -T    (lists rake tasks)
* rake jobs:work   (transcodes) 

running XSLT:
* rails runner bin/xslt.rb [xml file] [xsl file]
* rails runner bin/validate.rb [xml file] [format in Toolbox/Elan/Transcriber/Eopas]
* rails runner bin/transcode.rb [xml file] [format in Toolbox/Elan/Transcriber/Eopas]
